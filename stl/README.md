# 3D Print Files

STL/3MF files for the OpenChess board, pieces, and bases.

## Source

Official files from [MakerWorld — OpenChess Smart Chess Board](https://makerworld.com/en/models/1256302-openchess-smart-chess-board) by Concept_Bytes.

## Print Settings (from MakerWorld)

- Layer height: 0.2mm
- Walls: 2
- Infill: 0%
- Printer: Bambu Lab (or any FDM with ~250mm bed)

## Contents

- Chessboard tiles (64 squares) — snap-together, LED-friendly
- Frame & base plates — aligns with PCB mounting holes
- Chess pieces (32 total) — Staunton-style, with magnet insert slots
  - Available in 8×2mm and 10×2mm magnet versions

## Magnet Slots

Check which version of pieces you print:
- **8×2mm** — lighter hold, pieces don't attract each other
- **10×2mm** — stronger hold, but pieces may pull together
